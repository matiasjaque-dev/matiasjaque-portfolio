## 🚀 Cómo ejecutar localmente

1️⃣ Clonar el repositorio:

```bash
git clone https://github.com/matiasjaque-dev/matiasjaque-portfolio.git
```

2️⃣ Instalar dependencias:

```bash
npm install
```

3️⃣ Ejecutar en desarrollo:

```bash
npm run dev
```

4️⃣ Abrir en navegador:

```bash
http://localhost:5173
```
